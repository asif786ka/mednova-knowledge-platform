# Service Description — Operational Insights Service

**Client:** MedNova Solutions (internal operations)
**Project:** Operational Insights Service

## Summary
The Operational Insights Service provides data-driven operational support: dashboards and
forecasts for hospital bed capacity and staffing. It uses AI automation for demand forecasting.

## Technologies
The Operational Insights Service uses **Python**, **Prometheus** for metrics, and **Grafana** for
dashboards. It is deployed on **AWS**.

## Requirements
- Daily forecast refresh via a background queue.
- Observability with Prometheus and Grafana.
